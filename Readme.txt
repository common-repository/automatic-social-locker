Automatically opens the popup ask user clicks the Like, Plus or Tweet button to view your page's content.

== Description ==

This plugin enables you automatic lock content from users until they Like, Plus or Tweet your desired URL.

Plugin options:

- Choose to apply on all pages, all posts or select individually the pages you want
- Define your desired URL
- Show/Hide countdown timer
- Show/Hide close button

== Installation ==

- Go to your WordPress admin panel,  click on Plugin » Add New,  then click on the Upload Tab.
- Browse the file and upload automatic-social-lock.zip (zip files only)
- Click on Activate the plugin, now you have installed the WordPress plugin. You would have to configure the settings to fit your needs.
